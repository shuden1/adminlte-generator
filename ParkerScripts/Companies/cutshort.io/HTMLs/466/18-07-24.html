Frontend Developer (React, NextJS, Tailwind)

at StrategyConnect
Posted by Teja Thota
Remote only
1 - 8 yrs
₹3L - ₹7L / yr (ESOP available)

Javascript

+4 more

Location: Fully remote 

Company: StrategyConnect - Dubai based startup backed by Techstars 


As a Front-end Developer, you will be responsible for building user-friendly, responsive, and visually appealing web applications. You will work closely with our design team and senior developers to create high-quality web applications using modern front-end technologies such as React. 



Key Responsibilities: 

  • Build user-friendly, responsive, and visually appealing web applications using React, NodeJs and Tailwind. 
  • Collaborate with the design team to turn mockups into pixel-perfect web applications. 
  • Implement designs and create reusable UI components using React and Tailwind. 
  • Write clean, maintainable, and efficient code. 
  • Debug and troubleshoot issues as they arise. 
  • Ensure the application is optimized for performance, scalability, and usability. 
  • Continuously learn and keep up-to-date with the latest front-end development trends and technologies. 


Requirements: 

  • Strong proficiency in React, NextJS, HTML, CSS, and JavaScript. 
  • Experience with modern front-end development tools such as Webpack, Babel, and Git. 
  • Familiarity with Agile development methodologies. 
  • Excellent problem-solving skills and attention to detail. 
  • Ability to work collaboratively in a remote team environment. 
  • Excellent communication skills in English. 


If you are passionate about front-end development, have a strong focus on React, and want to work with a dynamic team of talented individuals, please apply to this exciting Front-end Developer position today!
