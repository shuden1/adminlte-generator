Открытые вакансии

Архитектор
Требования
  • опыт работы в роли архитектора - от 2 лет
  • опыт проектирования архитектуры на уровне микросервисов и компонентов
  • знание паттернов синхронного и асинхронного взаимодействия, оркестровки и хореографии
  • знание / опыт работы с Java-стеком
  • опыт работы в одной из СУБД: PostgreSQL, MS SQL, Oracle
Обязанности и задачи
  • проектирование архитектуры приложения (микросервисы) и интеграционных сервисов
  • проработка концептуальной и прикладной архитектуры
  • разработка, участие в разработке и согласовании проектной документации
  • контроль соответствия разработанных систем архитектурным требованиям
  • работа по гибким методологиям
Системный аналитик (TeamLead)
Требования
  • опыт работы системным аналитиком от трех лет;
  • наличие управленческого опыта будет преимуществом;
  • практический опыт работы по ГОСТ 34, 19;
  • высшее образование в ИТ, в идеале инжиниринг информационных систем;
  • навыки общения с бизнес-заказчиками и формализации требований;
будет плюсом
  • опыт ведения требований в Enterprise Architect;
  • опыт работы с EA, Jira и Confluence;
  • опыт проектирования интерфейсов с помощью Figma;
  • представление о требованиях по защите персональных данных и аспектов обеспечения безопасности.
Обязанности и задачи
  • взаимодействие с заказчиком;
  • курирование процесса аналитики на проектах;
  • распределение задач по аналитикам команды проекта;
  • выявление требований и формирование концептуальных решений;
  • проектирование информационных систем;
Системный аналитик
Требования
  • опыт работы в роли системного аналитика не менее 3х лет
  • хорошее знание и опыт работы с SQL
  • навык графического описания бизнес-процессов и интеграционных взаимодействий, например, в виде BPMN или UML диаграмм
  • опыт проработки интеграций на SOAP или REST-запросах
  • знание принципов построения UI
будет плюсом
  • опыт ведения требований в Enterprise Architect
  • представление о требованиях по защите персональных данных и аспектов обеспечения безопасности
Обязанности и задачи
  • сбор, анализ и документирование бизнес и функциональных требований заказчика
  • разработка и согласование технических заданий и проектных решений
  • постановка задач разработчикам
  • приемка разработанного программного обеспечения перед проведение приемо-сдаточных испытаний
  • организация и проведение приемо-сдаточных испытаний и ввода в эксплуатацию
Java разработчик (TeamLead)
Требования
  • опыт работы Java Developer от трех лет;
  • наличие управленческого опыта будет преимуществом;
  • понимание принципов работы JVM;
  • опыт коммерческой разработки на Java не менее 2-3 лет;
  • опыт использования Spring Framework (Boot, MVC, Data, Security);
Обязанности и задачи
  • проведение технической оценки спецификаций и постановок задач;
  • организация работы команды;
  • наставничество для junior/middle разработчиков своей команды;
  • формирование или принятие участия в формировании архитектурных решений;
  • разработка и интеграция новых сервисов в общую экосистему;
Java разработчик
Требования
  • уверенное владение Java и понимание принципов работы JVM
  • опыт коммерческой разработки на Java не менее 3х лет
  • опыт использования Spring Framework (Boot, MVC, Data, Security)
  • опыт работы с многопоточностью и асинхронностью
  • умение писать и оптимизировать SQL-запросы
будет плюсом
  • опыт работы с фреймворками и инструментами автоматизированного тестирования
Обязанности и задачи
  • принятие постановок задач от аналитиков
  • техническая оценка требований, участие в выборе деталей реализации
  • разработки и отладка сервисов
  • разработка и отладка интеграций сервисов
  • написание тестов
