Hey, this is Blake and Dallin from Buster.

Buster is a developer tool for training Large Language Models (LLMs) to write production-ready SQL queries for your database. Buster provides the workspace for fine-tuning, model evaluations, managing access controls, and more.

problem
  • Getting an LLM to understand the nuances of your schema is a nightmare
  • Pushing LLMs to production is a nightmare
  • Monitoring LLMs in production is a nightmare
solution
  • Train an LLM to understand database nuances
    • Buster auto-generates documentation for your entire database
    • Buster utilizes a training pipeline to build out domain knowledge around your database
    • Buster fine-tunes an LLM, specific to your database, to recognize query patterns and understand nuances
  • Push LLMs to production
    • Buster hosts models for you
    • Buster includes a UI for the LLM that can be embedded anywhere as an iFrame (users can build dashboards/reports/etc using natural language)
  • Monitor and improve performance
    • Buster provides a monitoring space for engineers to track performance, usage, conversations, sql generation, user feedback, etc
    • Buster provides tooling to debug and quickly fix performance gaps
how it works

  1. Auto-generate documentation for your schema

  2. Train your model

  3. Embed “chat with your data” capabilities into your product

ask

We’re live and have customers using Buster in production! You can request free access at buster.so

message for the skeptics

Many of you have been let down by text-to-sql solutions, that’s why we’re building Buster. If you’re a skeptic, we’re happy to connect you with current customers that use Buster in production :)
