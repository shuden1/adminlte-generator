Front-End Angular Developer

About the Role:

We are seeking an experienced Front-End Angular Developer with expertise in Angular 8+, RxJS, Material Design, and Tailwind CSS.

The ideal candidate will play a pivotal role in designing and implementing user interfaces that deliver seamless and visually appealing experiences.

Key Responsibilities:

  • Design and develop intuitive and responsive web applications using Angular 8+.
  • Implement complex UI features with a focus on user experience.
  • RxJS Integration: Utilize RxJS to manage asynchronous operations, event handling, and state management within the application.
  • Material Design Implementation: Apply Material Design principles and components to create visually appealing and consistent user interfaces.
  • Tailwind CSS Styling: Use Tailwind CSS to streamline and optimize the styling process, ensuring a clean and maintainable codebase.
  • Code Quality and Maintenance: Write clean, modular, and maintainable code, following industry best practices. Conduct thorough code reviews to ensure code quality and adherence to coding standards.
  • Collaboration and Communication: Collaborate effectively with cross-functional teams, including back-end developers, UX/UI designers, and product managers.
  • Testing and Debugging: Implement unit tests and conduct debugging to ensure the functionality and performance of front-end components.

Qualifications:

  • Bachelor’s or Master’s degree in Computer Science, IT, or a related field
  • At least 5 years of experience in front-end web development with a strong focus on Angular 8+.
  • Proficiency in HTML, CSS, JavaScript, TypeScript, RxJS, and Angular CLI.
  • Experience with Material Design and Tailwind CSS.
  • Strong problem-solving and debugging skills.
  • Familiarity with version control systems (e.g., Git).

 

Preferred Qualifications:

  • Knowledge of front-end build tools (e.g., Webpack, Grunt).
  • Experience with unit testing frameworks (e.g., Jasmine, Karma).
  • Understanding of UX/UI design principles and usability concepts.
  • Familiarity with RESTful APIs and integrating front-end applications with back-end services.

This role offers the opportunity to work in a dynamic, challenging environment and make a substantial contribution to the growth and success of Superhog. If you are a results-driven professional with the qualities mentioned above, we encourage you to apply and join our team.

Information:

  • Start date: ASAP Full time
  • Salary TBC
  • Location: Remote London / Barcelona time zone

Contact: jobs@superhog.com

Mid-Level .NET Developer

About the Role:

We are seeking a skilled Mid-Level .NET Developer with expertise in Azure Cloud and a strong foundation in microservices architecture. The ideal candidate will be responsible for designing, developing, and maintaining robust and scalable applications that leverage both the .NET framework and Azure Cloud services.

Development and Implementation: Collaborate with cross-functional teams to design, develop, and deploy high-quality software solutions using the .NET framework. Implement microservices-based architecture for scalable and maintainable applications.

Azure Cloud Expertise: Leverage Azure services such as Azure App Service, Azure Functions, Azure SQL Database, Azure Cosmos DB, etc., to build reliable and highly available applications. Design, deploy, and manage cloud-based applications on the Azure platform.

Microservices Architecture: Develop, maintain, and optimize microservices, ensuring they align with best practices for scalability, maintainability, and performance.

Code Review and Quality Assurance: Conduct code reviews to maintain code quality, readability, and adherence to best practices. Implement and follow testing strategies, including unit tests, integration tests, and automated testing frameworks.

Collaboration and Communication: Collaborate effectively with team members, product managers, and stakeholders to deliver high-quality solutions. – Communicate technical concepts clearly and concisely to non-technical team members.

Qualifications:

  • Bachelor’s or Master’s degree in Computer Science or a related field. At least 5 years of experience in software development using the .NET framework.
  • Solid understanding of microservices architecture and cloud-native development.
  • Experience working with Azure Cloud services.
  • Proficient in C# and .NET Core.
  • Strong problem-solving and analytical skills.
  • Excellent communication and teamwork abilities.

Preferred Qualifications:

  • Familiarity with containerization and orchestration platforms (e.g., Docker, Kubernetes).
  • Knowledge of DevOps practices and tools.
  • Experience with CI/CD pipelines and automated deployment processes.
  • Understanding of API design and implementation.

Senior .NET Developer

About the Role:

We are seeking an experienced Senior .NET Developer with expertise in Azure Cloud and a strong background in functional programming. The ideal candidate will play a crucial role in designing, implementing, and maintaining complex solutions that leverage both the .NET framework and cloud services in Azure.

Design and Development: Lead the design and development of high-quality, scalable, and maintainable software solutions using the .NET framework and related technologies. – Architect and implement cloud-based solutions on the Azure platform

Functional Programming: Apply functional programming concepts to solve complex problems and improve system performance. – Utilize functional programming languages (e.g., F#, Haskell) to develop efficient and robust software components.

Azure Cloud Expertise: Design, deploy, and manage cloud-based applications and services on the Azure platform. – Leverage Azure services like Azure Functions, Azure App Service, Azure SQL Database, Azure Cosmos DB, etc., to build resilient and highly available systems.

Microservices Architecture: Develop, maintain, and optimize microservices, ensuring they align with best practices for scalability, maintainability, and performance. 

Collaboration and Leadership: Collaborate effectively with cross-functional teams, including product managers, UX/UI designers, and other developers, to deliver high-quality solutions. – Mentor and provide guidance to junior developers, fostering a culture of continuous learning and improvement.

Performance Optimization: Identify and implement performance enhancements, ensuring optimal system response times and resource utilization.

Code Review and Quality Assurance: Conduct thorough code reviews to maintain code quality, readability, and adherence to best practices. – Implement and follow robust testing strategies, including unit tests, integration tests, and automated testing frameworks.

 

Qualifications:

  • Bachelor’s or Master’s degree in Computer Science or a related field.
  • 8 years of experience in software development using the .NET framework and related technologies.
  • Strong proficiency in functional programming languages and concepts. – Extensive experience working with Azure Cloud services.
  • Solid understanding of cloud architecture and best practices.
  • Demonstrated ability to lead and mentor a team of developers.
  • Excellent problem-solving and analytical skills.
  • Strong communication and interpersonal abilities.

 

Preferred Qualifications:

  • Familiarity with containerization and orchestration platforms (e.g., Docker, Kubernetes).
  • Knowledge of DevOps practices and tools.
  • Experience with CI/CD pipelines and automated deployment processes.
  • Understanding of API design and implementation.
